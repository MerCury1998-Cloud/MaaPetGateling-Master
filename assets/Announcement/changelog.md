# MaaPetGateling 更新公告

## v1.1.5 更新说明

### 新增功能

- **公会大厅自动化**
  - 自动化进入公会大厅
  - 自动化申请公会列表
  - 自动化审批公会成员

- **公会科技自动化**
  - 自动化研究科技树
  - 自动化升级科技等级
  - 支持多种资源捐献

- **公会BOSS挑战**
  - 自动化攻击公会BOSS
  - 支持多种攻击类型（弓剑/物理/超能）

### 优化改进

- 优化截图识别效率
- 新增多个场景识别图片素材
- 完善任务流程配置
